# qietuzai-cli

**本机安装环境:node git**

### 使用方法
```
npm install qietuzai-cli -g
```


### 初始化项目
```
Qietuzai init
```

### 选择一个项目模板
目前只有针对两种不同客户端的模板(手机移动端和传统pc端)
* touch
* pc
    
### 进入项目目录    
```
cd  your project_name
```

### install
```
npm install
```

### 在浏览器里预览
```
npm run server
```

### 产出页面(交付给小伙伴)
```
npm run build
```
