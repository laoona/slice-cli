# slice-cli

**切图仔的日常 - 生成日常切图所需项目目录 本机安装环境依赖: node git**

```
├── src
│   ├── js
│   └── images
│   └── scss
│   └── fonts
│   └── jpegs
│   └── psd
└── index.html
```

### 使用方法
```
npm install slice-cli -g
```


### 初始化项目
```
slice init
```

### 选择一个项目模板
目前只有针对两种不同客户端的模板(手机移动端和传统pc端)
* touch
* pc
    
### 进入项目目录    
```
cd  your project_name
```

### install
```
npm install
```

### 在浏览器里预览
```
npm run server
```

### 产出页面(交付给小伙伴)
```
npm run build
```
